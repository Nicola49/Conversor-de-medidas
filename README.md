Este projeto é o início de uma ideia pessoal, e pode chegar a ser um aplicativo ou site.

O programa funciona assim:

CASO "MOEDAS":

- Digite o número que represente a opção de medida que você quer;
- Digite o número que represente a MOEDA que você quer converter;
- Digite o VALOR a ser convertido;
- Digite o número que represente a MOEDA vai ser convertido pela primeira;
- Terminal mostrará o resultado da conversão;
